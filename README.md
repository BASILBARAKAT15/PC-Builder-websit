# PC Builder

موقع ويب مخصص لبناء وتجهيز أجهزة الكمبيوتر، يجمع بين عرض المنتجات، اختيار القطع، وإدارة الطلبات ضمن تجربة واحدة للمستخدم والإدارة.

## ما الذي يقدمه النظام؟

- تصفح مكونات الكمبيوتر حسب الفئات مثل المعالج، كرت الشاشة، اللوحة الأم، الذاكرة، التخزين، التبريد، ومزود الطاقة.
- إنشاء تجميعة PC مخصصة من خلال اختيار القطع وحفظها أثناء التصفح.
- التحقق من توافق المكونات المختارة باستخدام تحليل ذكي يساعد المستخدم على اكتشاف التعارضات أو الملاحظات المهمة.
- اقتراح تجميعة كاملة بالذكاء الاصطناعي بناءً على الميزانية ونوع الاستخدام.
- مساعد محادثة ذكي يجيب عن أسئلة المستخدم المتعلقة بتجميعات الكمبيوتر واختيار القطع.
- إضافة المنتجات إلى السلة، تطبيق كوبونات الخصم، وإتمام عملية الطلب.
- إنشاء حساب للمستخدم وتسجيل الدخول وإدارة الملف الشخصي وتغيير كلمة المرور.
- حفظ المنتجات في المفضلة ومزامنتها مع الحساب.
- عرض الطلبات السابقة وتتبع حالة الطلب للمستخدم.

## جانب الإدارة

- لوحة تحكم لإدارة المنتجات وإضافتها أو حذفها.
- إدارة المستخدمين والطلبات وتحديث حالة الطلبات.
- إدارة الكوبونات وتفعيلها أو إيقافها.
- عرض بيانات تشغيلية أساسية تساعد على متابعة نشاط المتجر.

## فكرة المشروع

المشروع يجمع بين متجر إلكتروني لمكونات الكمبيوتر وأداة عملية لمساعدة المستخدم على بناء تجميعة مناسبة لاحتياجه، مع دعم ميزات ذكية لتحسين الاختيار وتقليل أخطاء التوافق بين القطع.
